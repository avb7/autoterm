### Installation

```bash
# Quick install
./install.sh

# Or with make

```

### Basic Usage

**ZSH (interactive mode):**
```bash
# find all python files
[Press Tab Tab]
```

**Bash/Fish (command mode):**
```bash
autoterm find all python files
```
